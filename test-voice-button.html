Button mounted inside a specific container element.

Test 2: Custom Button Text

Button with custom labels for each state.

<script src="dist/olleh-voice-button.js"
  data-olleh-idle-text="Speak with Tax Expert AI"
  data-olleh-loading-text="Dialing..."
  data-olleh-active-text="Hang Up"
  data-olleh-container-id="custom-text-btn"></script>

Test 3: Custom Inline Styles

User overrides default look entirely via data-olleh-button-style.

<script src="dist/olleh-voice-button.js"
  data-olleh-button-style="background:#16a34a;border-radius:24px;padding:16px 32px;font-size:18px;"
  data-olleh-container-id="styled-btn"></script>

Test 4: All Customizable Attributes (KIM Demo)

Complete showcase of every available customization option:

  • Text Customization: All three button states (idle/loading/active)
  • Styling: Custom inline styles, CSS classes, width & height
  • Behavior: Extended agent timeout (60 seconds)
  • Placement: Container-based positioning
<div id="full-demo-btn"></div>
<script src="dist/olleh-voice-button.js"
  data-olleh-client-token="YOUR_TOKEN"
  data-olleh-container-id="full-demo-btn"
  
  data-olleh-idle-text="🎙️ Start Voice Consultation"
  data-olleh-loading-text="⏳ Connecting to Expert..."
  data-olleh-active-text="📞 End Consultation"
  
  data-olleh-button-style="background:linear-gradient(135deg,#8b5cf6,#6366f1);box-shadow:0 8px 24px rgba(99,102,241,0.4);font-weight:700;letter-spacing:0.5px;font-size:15px;border-radius:25px;width:200px;height:54px"
  data-olleh-button-class="custom-corporate-btn"
  
  data-olleh-agent-timeout="60000"
></script>

📋 Attributes Demonstrated: All 11 customizable attributes including client-token, container-id, all text states (idle/loading/active), styling options (inline + class + dimensions), and timeout behavior.
